## 💡 Example: Arithmetic Operators

```python
# Addition
print(98 + 76)  # 174

# String concatenation
print('Smith is earning ' + str(9800))  # Smith is earning 9800

# List concatenation
print([10, 20, 30] + [40, 50, 60])  # [10, 20, 30, 40, 50, 60]

# Floor Division
print(78 // 34)  # 2

# Modulus (remainder)
print(7689 % 10)  # 9 (last digit)

# Exponentiation
print(2 ** 3)  # 8
```
